# Telco-Customer-Churn
# 📊 Telco Customer Churn Prediction | توقع انفضاض العملاء

مشروع متكامل لتعلم الآلة (End-to-End ML) لبناء نموذج يتنبأ باحتمالية مغادرة العملاء لشركة الاتصالات بناءً على سلوكهم وفواتيرهم ونوع عقودهم.

## 🛠️ تقنيات المشروع (Environment)
- **بيئة العمل:** محلياً بالكامل (Linux Mint) لضمان خصوصية البيانات.
- **اللغة والحزم:** Python 3.x (`pandas`, `numpy`, `scikit-learn`, `matplotlib`, `seaborn`).

## 📋 مراحل العمل الأساسية
1. **EDA & التنظيف:** معالجة الفراغات النصية في `TotalCharges` وفهم نسب الانفضاض.
2. **Feature Encoding:** تحويل المتغيرات النصية إلى أرقام باستخدام `One-Hot Encoding`.
3. **بناء النموذج:** تدريب نموذج الغابة العشوائية `Random Forest Classifier` مع موازنة البيانات `class_weight='balanced'`.
4. **التقييم:** التركيز على مقياس الاستدعاء `Recall` لاكتشاف أكبر عدد من المغادرين.

## 💡 أهم النتائج والتوصيات (Insights)
- **نوع العقد:** العملاء ذوو العقود الشهرية (`Month-to-month`) هم الأكثر عرضة للمغادرة.
- **مدة البقاء:** العملاء الجدد في أشهرهم الأولى يحتاجون إلى برامج ترحيبية ودعم مكثف لمنع انفضاضهم.
- **التكلفة:** الفواتير الشهرية المرتفعة تدفع العميل للبحث عن بدائل، مما يتطلب تقديم عروض مرنة قبل المغادرة.
